# GlowUps Frontend — Design Demo

> **Note:** This is a visual demo only. No backend integration required.

## Run Instructions

```bash
cd frontend
npm install
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## What's Included

- **Palette:** cream / beige / gold / copper / bronze color tokens
- **Fonts:** Merriweather (headings) + Inter (body) via Google Fonts
- **Hero section** with large Merriweather heading and gold CTA button
- **3-up responsive expert card grid** (Sarah, Jessica, Maria — hardcoded demo data)

## Tech

- React 17 + react-scripts
- Tailwind CSS v2
- CSS custom properties for design tokens
