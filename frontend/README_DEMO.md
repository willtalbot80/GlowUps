# GlowUps — Frontend Demo

> **This is a static visual mock.** No backend APIs are called; all data is hardcoded.

## Running the demo

```bash
cd frontend
npm install
npm start
```

The app will open at <http://localhost:3000>.

## Pages

| Route | Description |
|-------|-------------|
| Home  | Landing page with hero section and feature cards |
| Experts | Static list of 3 example expert cards |

## Notes

- Styles use the project's Tailwind color palette: `cream`, `beige`, `gold`, `copper`, `bronze`.
- Navigation is handled with local React state (no react-router required).
- This scaffold was created as part of the `demo/frontend-ui` PR to preview the app design.
